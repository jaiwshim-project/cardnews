# AI CardNews Generator 메뉴 구성서

## 1. 전체 메뉴 구조 개요

본 시스템은 **사용자(User)** 와 **관리자(Admin)** 두 가지 역할로 분리되며,
각각 독립된 메뉴 체계를 갖습니다.

```
┌─────────────────────────────────────────────────────────────┐
│                    AI CardNews Generator                     │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   [사용자 영역]                    [관리자 영역]              │
│                                                              │
│   ┌─────────────┐                 ┌─────────────┐           │
│   │   홈        │                 │  대시보드    │           │
│   ├─────────────┤                 ├─────────────┤           │
│   │ 카드뉴스    │                 │ 카드뉴스    │           │
│   │  생성       │                 │  관리       │           │
│   ├─────────────┤                 ├─────────────┤           │
│   │ 미리보기    │                 │ 품질       │           │
│   │  & 다운로드 │                 │  모니터링   │           │
│   ├─────────────┤                 ├─────────────┤           │
│   │ 내 카드뉴스 │                 │ A/B 테스트  │           │
│   ├─────────────┤                 ├─────────────┤           │
│   │ 내 정보     │                 │ 도메인 규칙 │           │
│   └─────────────┘                 ├─────────────┤           │
│                                   │ 템플릿 관리 │           │
│                                   ├─────────────┤           │
│                                   │ 사용자 관리 │           │
│                                   ├─────────────┤           │
│                                   │ 시스템 로그 │           │
│                                   └─────────────┘           │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 2. 사용자(User) 메뉴 구조

### 2.1 메뉴 트리
```
[사용자 메뉴]
│
├── 홈 (/)
│   └── 서비스 소개 및 빠른 시작
│
├── 카드뉴스 생성 (/create)
│   ├── 입력 폼
│   ├── 생성 진행
│   └── 결과 확인
│
├── 미리보기 & 다운로드 (/preview/{id})
│   ├── 카드별 미리보기
│   ├── 카드 수정
│   └── 다운로드 옵션
│
├── 내 카드뉴스 (/my-cardnews)
│   ├── 생성 이력
│   ├── 상세 보기
│   └── 재사용/복제
│
└── 내 정보 (/profile)
    ├── 프로필 수정
    ├── 비밀번호 변경
    └── 이용 통계
```
